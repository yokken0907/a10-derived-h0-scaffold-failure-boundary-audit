# A10由来H0スキャフォールド失敗境界監査

**Version:** `v0.1.1-public-landing-and-metadata-refresh`  
**Status:** v0.1.0 の claim-bounded failure-boundary audit を、公開入口・メタデータ・検索導線向けに整理した刷新版です。  
**Repository:** https://github.com/yokken0907/a10-derived-h0-scaffold-failure-boundary-audit  
**Project page:** https://yokken0907.github.io/a10-derived-h0-scaffold-failure-boundary-audit/

本リポジトリは、A10由来のハッブルテンション・スキャフォールドについて、どこで物理モデル化に失敗したのかを記録する監査アーカイブです。

本リポジトリは、ハッブルテンションの解決、Lambda-CDM / ΛCDM の置換・反証、観測的発見、実データlikelihood、MCMC posterior、evidence、Bayes factor、AIC/BIC比較、または対応するΛCDM基準比較を主張しません。

## 要約

このアーカイブは、失敗したが境界づけられた宇宙論スキャフォールドの記録です。A10由来のH0候補は、診断・引継ぎ用の素材としてのみ残りました。しかし、宇宙論的比較に必要な第一原理的背景膨張ブリッジ `E(z)` / `H(z)` は、監査対象内では確立されませんでした。

そのため、このチェーンは実データlikelihood、Lambda-CDM / ΛCDM reference smoke、posterior比較、evidence評価に進みませんでした。

したがって本リポジトリは、宇宙論の成果ではなく、**研究ガバナンス、失敗局在、主張境界の記録**として読むべきものです。

## 現在の状態

v0.1.1 は、新しい理論的・観測的主張を追加するものではありません。公開ページ、README、メタデータ、manifest、検索導線を整理するための非破壊的刷新です。

PDF本文は、元の v0.1.0 失敗境界監査記録として維持しています。

## 許可される主張

- A10由来H0スキャフォールド素材が分解・監査された。
- 候補proxy/scaffold素材は、限定された仮説・引継ぎ素材として残った。
- 第一原理的な物理 `E(z)` / `H(z)` ブリッジ方程式は確立されなかった。
- そのため、Lambda-CDM / ΛCDM reference smoke、実データlikelihood、MCMC、posterior比較、evidence評価は実行されなかった。
- 結果は観測宇宙論の勝利・敗北ではなく、失敗境界の記録である。

## 禁止される主張

- A10がハッブルテンションを解決した。
- A10がLambda-CDM / ΛCDMを置換・反証・打倒した。
- A10がSN/BAO/CMB likelihoodで支持された。
- 物理的な `E(z)` / `H(z)` 方程式が導出された。
- このアーカイブが新しい観測宇宙論結果を確立した。
