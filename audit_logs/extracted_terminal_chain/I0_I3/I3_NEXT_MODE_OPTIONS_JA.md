# I3: Next-mode options

## 現在の推奨

この I0–I3 で一度、人間可読の terminal dossier として閉じてよい。

## 続ける場合の選択肢

### J0: repository / public handoff packaging

人間・専門家向けの最小公開パッケージに整える。論文主張ではなく、failure-localization dossier として扱う。

### K0: first-principles derivation restart from external theory

Friedmann equation、effective fluid、modified background、または phenomenological deformation のいずれかを外部理論側から再構成する。これは現行A10チェーンの延長ではなく、理論再設計である。

### STOP

ここで停止しても、雑な撤退ではなく、破綻位置と残存物を固定した bounded closeout である。

## 非推奨

D2 / ΛCDM reference smoke へ直接進むことは非推奨である。physical E(z)/H(z) が未成立だからである。
