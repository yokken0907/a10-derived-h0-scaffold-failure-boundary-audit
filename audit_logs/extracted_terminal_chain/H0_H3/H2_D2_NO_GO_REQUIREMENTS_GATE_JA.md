# H2: D2 no-go requirements gate

D2へ進むには、最低限以下が必要である。

1. physical E(z) の導出または明示的な背景方程式
2. H(z)=H0E(z) の単位・正規化契約
3. E(0)=1、E(z)>0、有限性の確認
4. distance-like integral ではなく observable distance へ接続する契約
5. h0_target_label が結果へ漏れていないことの確認
6. null-control と primary candidate の分離
7. 実データ asset / parser / adapter の準備
8. matched LCDM baseline の比較契約

現行成果物では、1 と 4 が未成立であるため、D2_READY=False とする。
