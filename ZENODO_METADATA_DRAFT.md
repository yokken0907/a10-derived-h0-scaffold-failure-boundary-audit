## Description

A10-Derived H0 Scaffold Failure-Boundary Audit v0.1.1 is a claim-bounded failure-localization and public metadata refresh of an A10-derived Hubble-tension scaffold archive.

The archive records a negative but informative result. Candidate proxy/scaffold material survived only as bounded handoff material, while the first-principles physical background-expansion bridge required for cosmological comparison, `E(z)` / `H(z)`, was not established inside the audited artifacts. Because that bridge was missing, the chain stopped before Lambda-CDM reference smoke, real-data likelihood testing, MCMC, posterior comparison, evidence metrics, Bayes-factor evaluation, AIC/BIC comparison, or any Hubble-tension solution claim.

This v0.1.1 release does not introduce new cosmological, theoretical, or observational claims beyond the original v0.1.0 audit. Its main purpose is to improve public readability, GitHub Pages compatibility, metadata consistency, manifest verification, and discoverability.

The archive should be read as a research-governance and failure-boundary record: it shows where a tempting Hubble-tension scaffold had to stop, and preserves the boundary so that the failed route is not later misread as a cosmological result.

This release does not claim that A10 solves the Hubble tension, replaces Lambda-CDM / ΛCDM, falsifies Lambda-CDM / ΛCDM, provides observational support, defines a physical `E(z)` / `H(z)` model, completes SN/BAO/CMB likelihood testing, or reaches a peer-reviewed cosmological conclusion.
