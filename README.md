This repository archives a claim-bounded failure-boundary audit of an A10-derived Hubble-tension scaffold.

It **does not** claim a solution to the Hubble tension. It **does not** claim a replacement for Lambda-CDM / ΛCDM. It **does not** report a real-data likelihood, MCMC posterior, evidence value, Bayes factor, AIC/BIC comparison, or matched Lambda-CDM baseline result.

## Plain-language summary

This archive records a failed but bounded cosmology-scaffold route. A candidate A10-derived H0 scaffold survived only as diagnostic/handoff material. The required first-principles physical background-expansion bridge, `E(z)` / `H(z)`, was **not established** inside the audited artifacts. Because that bridge was missing, the chain stopped before real-data likelihood testing, Lambda-CDM reference smoke, posterior comparison, or evidence evaluation.

The archive is therefore best read as a **research-governance and failure-localization record**: it documents where a tempting Hubble-tension scaffold had to stop, and preserves the boundary so that the failed route is not later misread as a cosmological result.

## Main result

The audited chain retained:

- a candidate proxy;
- an internal diagnostic curve;
- a contract-only scaffold;
- bounded hypothesis/handoff material;
- a failure-localization record.

However, the audited chain did **not** establish:

- a first-principles physical `E(z)` equation;
- a first-principles physical `H(z) = H0 E(z)` equation;
- an observable cosmological distance model;
- a SN/BAO/CMB likelihood;
- a matched Lambda-CDM / ΛCDM baseline comparison;
- MCMC, posterior, evidence, Bayes-factor, AIC, or BIC evaluation;
- a Hubble-tension solution claim;
- a Lambda-CDM / ΛCDM replacement claim.
